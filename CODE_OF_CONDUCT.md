# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Full text and enforcement guidelines are at the canonical URL above. The short version:

- Be respectful in issues, PRs, comments, and any other project space.
- Disagree on technical content, not on people.
- If someone behaves badly toward you or others, email `conduct@eidetic.works` (or open a private security advisory on this repo). Reports are handled confidentially.

The maintainers will enforce this Code of Conduct in good faith. Persistent or severe violations may result in being blocked from project spaces.

## Scope

This applies in all project-controlled spaces: issues, pull requests, discussions, the project Discord, and any official communication channel.
